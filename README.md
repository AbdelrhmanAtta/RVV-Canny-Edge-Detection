# Canny-Edge-Detection
In this project, you will implement a Canny edge detection pipeline targeting RISC-V (rv64gcv) and running on QEMU in user-mode emulation. Edge detection is a foundational operation in computer vision: it identifies the boundaries of objects in an image by finding regions where pixel intensity changes sharply.
